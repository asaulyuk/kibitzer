# Kibitzer ♞

Free open source chess trainer with Stockfish analysis and AI coaching.

**[▶ Play now](https://asaulyuk.github.io/kibitzer)**

## Features
- Import games from Lichess and Chess.com
- Stockfish eval bar on every position (via Lichess Cloud Eval)
- Play against the engine
- AI coaching and chat powered by Groq (free tier)
- Ukrainian UI 🇺🇦

## Setup
1. Get a free Groq API key at [console.groq.com](https://console.groq.com)
2. Open the app and paste your key in Settings ⚙
3. No credit card needed
